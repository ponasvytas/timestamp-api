TImestamp microservice

Enter date in URL to get unix and natural format

Example
+ https://timestamp-ponasvytas2.herokuapp.com//December%2015,%202015
+ https://timestamp-ponasvytas2.herokuapp.com/1450137600 +
Example output:
+ { "unix": 1450137600, "natural": "December 15, 2015" }
+ { "unix": null, "natural": null }
